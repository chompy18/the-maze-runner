##Regex Syntax
Consider the following statement containing regular expressions:

```javascript
var text = "testing: 1, 2, 3"; 
var pattern = /\d+/g;
```

In order to check if the pattern matches, the statement is

a) text==pattern
b) text.equals(pattern)
c) text.test(pattern)
d) pattern.test(text)
